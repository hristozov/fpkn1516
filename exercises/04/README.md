Разгледано на упражнението
==========================

* Итеративен процес за приближено изчисление на квадратен корен ([`sqrt`](01sqrt.scm), по [метода на Херон](http://en.wikipedia.org/wiki/Methods_of_computing_square_roots#Babylonian_method)).
* Процедури, които приемат процедури за аргументи. Анонимни ([lambda](http://www.schemers.org/Documents/Standards/R5RS/HTML/r5rs-Z-H-7.html#%_sec_4.1.4)) процедури.
* Процедури [`sum`](02sum.scm), [`sum-next`](03sum-next.scm) и [`product`](04product.scm). Изграждане на абстракция на „акумулиращ“ процес ([`accumulate`](05accumulate.scm)).
* Процедура `accumulate`. Задачи.
